# Automatic Fire Extinguisher Robot

Short description: Autonomous robot that detects and extinguishes Class A fires using flame sensors, Arduino Mega, motor driver, and a water pump.

## Contents
- `code/` — Arduino source
- `hardware/` — block & circuit diagrams
- `documentation/` — report and presentation
- `documentation/images/` — build and test photos

## Quick Start
1. Open `code/fire_robot.ino` in Arduino IDE.
2. Wire components per `hardware/circuit_diagram.png`.
3. Upload to Arduino Mega, power with appropriate battery pack.
